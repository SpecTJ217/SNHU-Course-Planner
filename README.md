Course Planner
Overview

Course Planner is a C++ application developed for CS 300 that helps academic advisors manage course information. The program loads course data from a CSV file, stores it in a binary search tree, prints an alphabetically sorted course list, and allows users to search for a specific course along with its prerequisites.

Reflection

This project strengthened my understanding of how data structures affect both performance and software design. Comparing vectors, hash tables, and binary search trees showed me that selecting the right data structure depends on the problem being solved. For the final project, I implemented a binary search tree because it provides efficient searching while naturally supporting alphabetical output through an in-order traversal.

One of the biggest lessons I learned was the importance of planning before writing code. Creating pseudocode first made implementation much more organized and helped identify potential issues early. Breaking the project into smaller functions also made debugging and testing much easier.

This course also reinforced the value of writing clean, modular, and maintainable code. Using descriptive function names, input validation, and reusable functions resulted in code that is easier to understand, modify, and expand. These are practices I will continue using in future coursework and software development projects.
